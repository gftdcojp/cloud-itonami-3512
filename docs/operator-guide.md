# Operator Guide

## First Deployment

1. Register one site and one meter.
2. Import historical consumption and generation data.
3. Run the advisor in read-only mode.
4. Compare recommendations with current operation.
5. Enable human-approved dispatch recommendations.

## Minimum Production Controls

- customer-owned telemetry store
- clear tariff and settlement source
- approval workflow for dispatch and billing
- incident contact for site operators
- monthly audit export

## Certification

Certified operators must prove telemetry provenance, settlement traceability,
and that automated actions cannot bypass the Grid/Policy Governor.
